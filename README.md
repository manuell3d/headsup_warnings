HeadsUp Warnings is a Blender add-on designed to help users avoid common mistakes by visually highlighting potentially destructive settings. 
It overlays important warnings in the 3D viewport and compositor, ensuring users stay informed about settings that might impact their workflow. 
With color-coded UI borders and direct on-screen messages, this add-on provides real-time feedback to help prevent accidental errors before they cause problems.

It comes with 45+ conditions to be warned about, each can be toggled via the Addon Preferences. 
With HeadsUp Warnings, you can work more confidently in Blender, reducing mistakes and improving efficiency.
